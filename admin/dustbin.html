Animation using Minecraft 1-3

Ages 8 - 13, Grades 1 - 3

In this class, younger students will learn how characters move and look alive in movies and video games! They will be introduced to basic 2D and 3D animation techniques using Minecraft characters and stages

Animation using Minecraft 3-7

Ages 8 - 13, Grades 3 - 7

In this class, older students will be introduced to basic 2D and 3D animation techniques using Minecraft characters and stages. Kids will expand their observation skills and imagination by studying motion in reality and how it influences motion in animation

Coding & Design Games with Scratch 1-3

Ages 8 - 13, Grades 1 - 2

In this class, students will learn basic programming concepts using a simple drag-and-drop interface within the software Scratch, a project developed by the MIT Media Lab.

Coding & Design Games with Scratch 3-7

Ages 8 - 13, Grades 3 - 7

In this course students will learn basic programming and logic flow and learn how to apply those concepts to create games. Using Scratch, a scripting language developed by MIT to give kids an easy entry into coding...

3D Game Design & Production 3-7

Ages 8 - 13, Grades 3 - 7

Students will learn how to create exterior environments, and apply different procedures of modeling and texturing. Students create their own outdoor terrain using advanced tools such as terrain sculpting, particle creation, and dynamic lighting..
